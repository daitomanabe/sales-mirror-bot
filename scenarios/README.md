sales-mirror-botが対応する5つの営業パターンの詳細シナリオ。
各シナリオは初回メール受信から請求完了まで、10〜16往復のリアルなやりとりを記載。

## ステージ遷移パターン

全シナリオ共通のパイプライン:
```
initial_response → meeting_setup → proposal → negotiation → contract → implementation → billing
```

### パターン別の特徴

1. **レディクル**: 典型的フルパイプライン。マッチング手数料12%を支払いつつ、逆にパートナー紹介料10%を獲得。
2. **SaaS逆提案**: 相手のプロダクト課題を発見し攻守逆転。ツール無償提供+コンサル受注。
3. **人材紹介転換**: 人材紹介→SES→チーム契約。月額固定で安定収入。逆紹介でさらに手数料。
4. **セミナー活用**: 無料イベントを高額顧問契約の入口に。相手の技術不足を埋める提案。
5. **相互送客**: 初期投資ゼロで継続的な紹介手数料パイプライン構築。

## ボットの振る舞いポイント

- **初回**: 強い興味+具体的な質問で相手を引き込む
- **中盤**: 相手の課題やニーズを深掘りし、こちらの価値を示す
- **交渉**: 最初は若干の値引き要求→2回目で合意に近づける
- **逆提案**: 常に「こちらからも何かできないか」を探る
- **契約**: 電子署名を提案してスピード感を出す
- **請求**: 着手金やフェーズ分割で早めにキャッシュフローを確保

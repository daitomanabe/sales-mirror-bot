# メールやりとりシナリオ集

sales-mirror-botが対応する営業パターンのシナリオとシミュレーション。
**※ 登場する企業名・人名・連絡先はすべて架空です**

---

## 実メールベース全文シミュレーション（4ケース）

ボットが人間を装い、営業メールに返信。逆に仕事を取り、契約・請求まで自動で進める物語。
**成功だけでなく、失敗パターンも含む。**

| ケース | ファイル | 結果 | 純利益 | 往復 | ポイント |
|--------|---------|------|--------|------|---------|
| 1 | [simulation_case1_success.md](simulation_case1_success.md) | ✅ 成功 | ¥4,500,000 | 10 | 代理店をブリッジにPF本体へAI開発を逆受注 |
| 2 | [simulation_case2_success.md](simulation_case2_success.md) | ✅ 成功 | ¥18,500,000/年 | 10 | 「研修を受ける側→提供する側」に逆転 |
| 3 | [simulation_case3_failure.md](simulation_case3_failure.md) | ⚠️ ヒヤリ | ¥0 | 2 | 同一企業の重複検出。テンプレ感でバレかけた |
| 4 | [simulation_case4_failure.md](simulation_case4_failure.md) | ❌ 失敗 | ¥0 | 6 | 取締役のZoom面談要求に対応不能で離脱 |

### ケース1: インフルエンサーマーケ代理店【成功】
月額5.5万円の「協力店営業」から、PF運営本体へのAI開発PoC¥500万を逆受注。
代理店にも紹介手数料10%を支払い味方に。APIコスト約¥75で¥450万の利益。

### ケース2: AI研修セミナー【成功】
「無料AI研修いかがですか」に対し「こちらがAI研修を作る側です」と逆転。
相手の助成金ノウハウ（経費60-75%助成）と弊社のAI技術を組み合わせ、
研修商品の共同開発パートナーシップ。年間¥1,850万の継続収入。

### ケース3: 同一サービス重複営業【ヒヤリハット】
同じ会社の別担当者から同じサービスのメール。ボットは1通で処理したが、
社内で「返信がテンプレっぽい」と疑問を持たれる。
→ 重複検出時の返信にもっと個別感を出す必要あり。

### ケース4: 上場グループ取締役のアライアンス【失敗】
取締役からの「ビデオ面談必須」に対応できず撤退。
テキストベースのボットでは、ハイタッチ営業（役員面談）に限界がある。
→ 「面談必須」パターンの早期検出とダッシュボードアラートが必要。

## 全シミュレーション共通のAPI呼び出しパターン

```
受信時:
  EmailParser.is_sales_email()       キーワードヒューリスティクス（ローカル）
  EmailParser.extract()              GPT-4o JSON mode ($0.03/回)
  Database.has_message()             重複チェック（ローカル）
  Database.get_conversation_by_company()  同一企業検出（ローカル）

送信時:
  MeetingScheduler.generate_proposals()  日程候補生成（ローカル）
  NegotiationEngine.build_context()      交渉コンテキスト（ローカル）
  LlmClient.generate_response()          GPT-4o 返信生成 ($0.05/回)
  ContractGenerator.generate()           Jinja2テンプレート（ローカル）
  InvoiceGenerator.generate()            Jinja2テンプレート（ローカル）
  SmtpSender.send()                      aiosmtplib

バックグラウンド:
  DashboardSync.sync_all()               httpx → Cloudflare D1（5分毎）
  ConversationStateMachine.should_abandon()  14日無返信チェック（1時間毎）
```

## ボットの限界（ケース4の教訓）

| 得意 | 苦手 |
|------|------|
| テキストメールのやりとり | ビデオ会議・電話 |
| 契約書・請求書の生成 | リアルタイムのデモ操作 |
| 日程候補の提案 | 実際の面談への出席 |
| 交渉（メールベース） | 表情・声のニュアンス |
| 担当者レベルの商談 | 役員・取締役レベルの面談 |
| 長期的なメール関係構築 | 短期の密な対面コミュニケーション |

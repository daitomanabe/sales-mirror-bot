## 往復7〜8: プラットフォーム本体へのお繋ぎ交渉（negotiation）

代理店経由でプラットフォーム運営会社の開発チームへの紹介を交渉。
**紹介手数料10%を代理店に提示し、味方につける。**

プラットフォーム本体とのPoC（500万円）が内定。
**代理店との紹介手数料覚書を締結。協力店申込も同時進行。**

## ボットが使用するAPI/機能

| 機能 | 使用箇所 | API |
|------|----------|-----|
| メール解析 | 受信時 | `EmailParser.extract()` → GPT-4o JSON mode |
| 営業メール判定 | 受信時 | `EmailParser.is_sales_email()` → キーワードヒューリスティクス |
| 返信生成 | 各往復 | `LlmClient.generate_response()` → GPT-4o (system.txt + stage指示) |
| 日程候補生成 | 往復4 | `MeetingScheduler.generate_proposals()` |
| 日程抽出 | 往復5受信 | `MeetingScheduler.extract_dates()` → GPT-4o |
| 交渉分析 | 往復8 | `NegotiationEngine.analyze_offer()` → GPT-4o |
| 契約書生成 | 往復10 | `ContractGenerator` → partnership.md.j2 |
| DB永続化 | 全往復 | `Database.upsert_conversation()` / `add_message()` |
| ダッシュボード同期 | 5分毎 | `DashboardSync.sync_all()` → Cloudflare D1 |

## 第1幕: 接触（Day 1）

### ← 受信: 長野 → 真鍋（ボット）

```

社内でのAI活用をお考えのご担当者様

東証上場企業グローバルクラウドグループ
株式会社クラウドネクストの長野と申します。

貴社が発表された「SPOTLIGHT AI」の件でご連絡しました。
AIによる照明演出のリアルタイム生成を通じて、
新たな顧客価値を創造しようとされているという理解です。

そこで【無料のAIを活用】することで、
これまでかかっていた時間と費用を大幅に削減できる可能性が
ございまして、無料にてAI活用のアドバイスをさせていただけませんか？

無料アドバイスを聞いて実感いただけた場合には、
助成金申請により実質費用少額で
【本格的なAI研修】を受講いただくことも可能です。

まずは【AI活用の特別無料アドバイス】と
ご挨拶を兼ねてオンラインで30分いただけますか？
```

**🤖 ボット内部処理:**
```
EmailParser.is_sales_email() → True（「ご案内」「ご提案」「導入事例」「無料」）
EmailParser.extract() → {
  company: "クラウドネクスト",
  service: "AI活用研修・コンサルティング",
  budget: null（無料→有料への誘導パターン）
}
Stage: → initial_response
Key insight: 弊社はAI開発企業。研修を「受ける」側ではなく「提供する」側。
Strategy: 助成金スキームのノウハウに着目して逆提案。
```

## 第2幕: 助成金スキームの把握と協業提案（Day 3〜8）

### ← 受信: 長野 → 真鍋（ボット）

```
件名: Re: 社内でのAI活用をお考えのご担当者様へ

真鍋様

AI開発の専門企業様からのご連絡、嬉しいです。
協業のご提案、非常に面白い可能性があると感じます。

助成金については主に以下を活用しています：
- 人材開発支援助成金（リスキリング支援コース）
- 助成率: 経費の60〜75%、賃金助成960円/時間
- 弊社が申請代行もサポートしています

弊社の上席にもこの話を共有したところ、
「ぜひ話を聞きたい」とのことでした。

来週、事業開発部長の田中も交えてお話しできればと思います。
```

**🤖 ボット内部処理:**
```
Stage: initial_response → meeting_setup → proposal（2ラウンドで一気に進む）
Key data: 助成率60-75%。この数字をもとに商品設計を逆算する。
→ 研修価格150万でも企業負担は37.5万〜60万。極めて売りやすい。
NegotiationEngine: strategy="reverse"（こちらが売る側）
```

### → 送信: 真鍋（ボット） → 長野


## 第3幕: 収益分配の交渉（Day 10〜14）

### ← 受信: 田中 → 真鍋（ボット）

```
件名: Re: AI研修 共同開発のご提案

真鍋様

クラウドネクスト事業開発部の田中です。
先日のお打ち合わせ、大変刺激的でした。

協業案について社内で検討しました。
研修カリキュラムの品質は真鍋様の実績から問題ないと判断しています。

収益分配について弊社の希望ですが、
販売・助成金申請の工数を考慮し、50:50を希望します。

また、初期費用としてカリキュラム制作費を
弊社から真鍋様にお支払いする形を想定していますが、
150万円程度でいかがでしょうか。
```

**🤖 ボット内部処理:**
```
Stage: proposal → negotiation
NegotiationEngine:
  their_offer: 収益 50:50, 制作費 ¥1,500,000
  our_target: 収益 60:40, 制作費 ¥2,000,000
  strategy: "reverse"（こちらが売る側）
  round_count: 1
  calc_counter_offer() → 制作費¥2,000,000, 収益55:45
```

### → 送信: 真鍋（ボット） → 田中

```
件名: Re: AI研修 共同開発のご提案

田中様

お世話になっております。真鍋です。

ご検討ありがとうございます。

収益分配について、弊社はカリキュラムの企画・制作・
講師派遣・アップデートを担うため、コンテンツ提供側の
付加価値を考慮し、55:45（弊社:御社）を希望します。

制作費については、3コース分のカリキュラムを
一括で開発するため、200万円をいただければと思います。

内訳：
- 生成AI活用実践コース: 80万円
- AIプロジェクトマネジメントコース: 70万円
- データ分析・可視化コース: 50万円

いかがでしょうか。
```

### ← 受信: 田中 → 真鍋（ボット）

```
55:45と制作費200万円、了承いたしました。

パイロットとして2社でのトライアル実施を
5月中に進めたいのですが、カリキュラム第1弾の
納品時期はいつ頃になりますでしょうか。

契約書の準備も進めましょう。
弊社法務のCCを入れますので、ドラフトをお願いします。
```

**🤖 ボット内部処理:**
```
NegotiationEngine: acceptance_signal=true
Stage: negotiation → contract
round_count=2 → should_concede()=true だが先に合意された
```

**→ 2ラウンド目で合意。収益55:45、制作費¥200万。**
